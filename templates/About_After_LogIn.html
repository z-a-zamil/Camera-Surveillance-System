Juba Camera Surveillance System

In our project, we will mainly detect faces using Python and OpenCV. We plan to build the backend of our project using python frameworks Django or Flask and frontend through JavaScript. Our initial plan for the database is to use SQLite. And we will use HTML, CSS, and React JS to build our front end. We will use OpenCV for human face detection. We plan to use the JavaScript framework React for the front end. First, we will collect the data of students and staff of the university through registration. We will store that data in the database. The machine will scan people's faces through the camera, take their information from the database, and display that information as output. The output will show Name, Department, and IDs next to their appearance. As a result, if someone is not associated with the university, he can be detected easily.
